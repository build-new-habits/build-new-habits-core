03. Design System & UX

Design Philosophy

Mobile-First: All layouts start at 320px width.

Accessibility First: WCAG Level AAA Compliance (High contrast, scalable text).

"Quiet" UI: Solid colors, opaque cards, no "glassmorphism" glitches. Reduced motion preferences respected.

Component Specs

The "Master Card" (Style B)

Header: Colored band at the top (Category Color) with Icon and Category Name.

Body: Solid Background (#1e293b).

Content: Title (H3), Meta Data (Time/Cost), Tags.

Interaction: Click anywhere to Flip.

Back Face: Detailed instructions + "Add to Plan" button.

Navigation

Bottom Bar: Fixed, 5-tab maximum.

Fab (Floating Action Button): "Quick Add" for high-frequency tasks (Log Spend, Log Food).

Typography

Headings: Inter (Weights: 600/800).

Data/Numbers: JetBrains Mono (For tabular alignment).
